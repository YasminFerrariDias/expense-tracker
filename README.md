# Expense Tracker

Aplicação web para controle de despesas, permitindo adicionar, visualizar e remover gastos, com atualização automática do total.

---

## 🚀 Funcionalidades

* Adicionar despesas com nome, categoria e valor
* Remover despesas da lista
* Atualização automática do total
* Contagem dinâmica de despesas
* Formatação de valores em Real (R$)

---

## 🧠 Conceitos aplicados

* Manipulação do DOM
* Eventos (submit e click)
* Criação dinâmica de elementos
* Trabalhando com arrays e objetos
* Formatação de moeda com `toLocaleString`
* Lógica de cálculo em tempo real

---

## 📂 Estrutura

* `index.html` → estrutura da aplicação
* `styles.css` → estilização da interface
* `scripts.js` → lógica da aplicação

---

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/yasminferraridias/expense-tracker.git
```

2. Abra o arquivo `index.html` no navegador
